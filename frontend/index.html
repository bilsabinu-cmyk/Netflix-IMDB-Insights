ABOUT THE PROJECT

Where Netflix Content Meets IMDb Intelligence.

Netflix & IMDb Insights is a Machine Learning powered web platform that combines Netflix catalogue information with IMDb audience sentiment to generate meaningful entertainment analytics. The platform enables users to search Netflix titles, visualize content trends, explore interactive dashboards, and predict whether audience sentiment towards a movie review is positive or negative using trained machine learning models.


The project integrates data engineering, data visualization, and predictive analytics into a single platform, providing a modern solution for understanding audience preferences and streaming content performance.
