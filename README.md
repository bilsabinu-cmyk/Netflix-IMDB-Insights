# 🎬 Netflix IMDb Insights

Netflix IMDb Insights is a data analytics project that explores Netflix content and IMDb ratings to uncover meaningful insights. The project includes data cleaning, exploratory data analysis (EDA), and interactive visualizations to better understand trends in Netflix movies and TV shows.

## 📌 Project Overview

This project analyzes Netflix titles along with IMDb ratings to answer questions such as:
- Which genres have the highest IMDb ratings?
- How has Netflix content evolved over the years?
- Which countries produce the most Netflix content?
- What are the trends in movies vs. TV shows?
- Which titles have the highest IMDb ratings?

## 🚀 Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Interactive visualizations
- IMDb rating analysis
- Genre and country-wise insights
- Year-wise content trends

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/bilsabinu-cmyk/Netflix-IMDB-Insights.git
```

2. Navigate to the project folder

```bash
cd Netflix-IMDB-Insights
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

4. Open the Jupyter Notebook and run the analysis.
